xptr by glro

Pure JavaScript XPointer Implementation

The goal of xptr is to provide a pure JavaScript implementation of the XPointer standard. This project also aims to develop a backing extensible JavaScript XPath implementation, which is able to handle the XPointer extensions. The current state of this project is non-functional. An LL(1) parser generator has been written (Lligen), along with an XPath parser (written as a Lligen grammar) and hand-coded lexer. The parser will return an Abstract Syntax Tree for the XPath expression and a couple of "pretty printers" are written to rewrite the AST. The next priority is writing the XPath interpreter.
